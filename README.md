# JavaScript Fundamentals Toolbox

A collection of custom JavaScript functions, logic challenges, and DOM scripts demonstrating my progress in mastering core fundamentals.

## Project Structure

- `string-methods/`: Contains utility scripts for manipulating and transforming string data.
- `arrowMath.js`: Scripts showcasing mathematical operations using modern arrow function syntax.
- `index.js`: A primary entry point for testing miscellaneous logic.
- `stringReverse.js`: A reusable utility function for reversing string input.
- `toolboxCheck.js`: A sandbox environment for testing new concepts and regex patterns.

## Purpose

This repository serves as a live record of my development journey, focusing on clean code, Git version control, and modular programming.
